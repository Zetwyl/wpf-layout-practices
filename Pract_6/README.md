### Stage 1
###### Task 1
При указании несуществующего ElementName в окне 'Вывод' (Output) появляется ошибка System.Windows.Data Error: 4. Она сообщает, что источник привязки не найден (Cannot find source). Визуально это проявляется так: элемент-приемник (TextBlock) не реагирует на изменение источника и использует значение шрифта по умолчанию. Приложение при этом не аварийно не завершается
###### Task 2
Различие в поведении элементов обусловлено стандартными значениями свойства UpdateSourceTrigger. Для Slider это PropertyChanged (мгновенно), а для TextBox — LostFocus (при выходе из поля). Изменение Mode на TwoWay влияет только на двунаправленность связи, но не на частоту обновления данных
###### Task 3
Для реализации полной синхронизации элементов управления был использован механизм изменения триггера обновления привязки. Стандартное поведение TextBox (обновление по потере фокуса) не обеспечивало мгновенного соответствия между введенным числом и размером шрифта TextBlock. Использование UpdateSourceTrigger=PropertyChanged позволило связать элементы напрямую в момент изменения текста, что полностью выполняет условие задачи о соответствии значений друг другу
